About Myself

My Life So Far

I am an aspiring Front-End Developer with a strong focus on creating modern, responsive web applications. My journey into technology began early, fueled by a fascination with how things work. From building my first PC to experimenting with tools like Adobe Photoshop, I discovered a passion for blending creativity with technical skills.

Currently, I am focused on web development. This website is my first Front-End creation, designed and built from the ground up using HTML, CSS, and a touch of JavaScript. Aside from this website I have developed two React-based applications using TypeScript: a To-Do App and a Weather App that utilizes APIs for real-time data fetching.

Outside of coding, I freelance as a graphic designer, which has sharpened my eye for design and user experience. I also hold a Python certification from Cisco. Python was the first programming language I learned.

You can explore my projects and other work in the Projects section of this site. I am excited to take the next step as a Front-End Developer, combining my skills and creativity to craft engaging user experiences.
